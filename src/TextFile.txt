Here’s what you should do to allow public access to port 8080:

go to your security group settings in your cloud provider's console.
go to the "Inbound rules" tab.
1. Add a New Rule
Click on the "Add rule" button.

2. Configure the Rule
Type: Select Custom TCP from the dropdown.

Port range: Enter 8080.

Source:

For public access, choose Anywhere-IPv4 (which fills in 0.0.0.0/0).

You can also manually select Custom and type 0.0.0.0/0 in the box.

Description (optional): You can enter something like "Allow API access".

3. Save the Rule
Click the orange "Save rules" button
